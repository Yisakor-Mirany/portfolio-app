# React Counter Demo

## Overview
This project is a simple React application built with Vite. It demonstrates key concepts in React including state management, asynchronous updates, batching, and the idea of state as a snapshot.

## Features
- Increment count by 1
- Increment count after a 2-second delay
- Demonstrates incorrect double increment due to batching
- Demonstrates correct double increment using updater functions
- Reset button for testing

## Concepts Demonstrated
- useState Hook
- State as a snapshot
- React batching behavior
- Asynchronous state updates with setTimeout
